About this dashboard

This version is designed to be stable and readable. The counters update every second from one baseline and one growth rate, instead of jumping with random values. The maps use fixed activity zones so the page stays clean and reliable.

The numbers below are simulation estimates built from a starting mileage total, a miles-per-day growth assumption, and a comparison between a human crash rate and an FSD crash rate. This keeps the dashboard moving in a smooth and believable way.
